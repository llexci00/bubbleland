# 🫧 버블랜드 (BubbleLand)

아늑한 픽셀아트 웹 앱. 방·거실·부엌을 꾸미고, 공원과 상점을 오가며, 컴퓨터로 다이어리·운세·아케이드를 즐기는 코지 게임.

- **바닐라 HTML/CSS/JS** — 빌드 도구·프레임워크 없음. `index.html`만 열면 실행돼요.
- 실시간 시간·날씨·계절, 코인 경제, 꾸미기·스킨·방 테마, 곰돌이 점집(수정구 점보기), 스네이크 아케이드 등.

## 실행

- **바로 열기**: `index.html`(또는 자체완결형 `bubbleland.html`)을 브라우저로 열기
- **로컬 서버**: `npx serve` 등으로 폴더를 서빙

## 단일 파일 빌드

외부 CSS/JS를 전부 인라인한 한 파일(`bubbleland.html`)을 다시 만들려면:

```bash
node build.js
```

## 조작

말풍선(할일)을 눌러 완료 → 코인 획득 · 상단 아이콘으로 방 이동/소리 · `꾸미기`로 배치와 스킨 변경 · `밖으로 나가기`로 공원/상점.
